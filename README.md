# speech-to-text-whisper

офлайн-инструмент для локальной расшифровки речи прямо в браузере. никаких внешних серверов, апи-ключей и слива данных. всё крутится на клиентской стороне через transformers.js и ONNX Runtime Web.

## фичи

- приватность: аудио не загружается в интернет. модель скачивается один раз в кэш браузера, после чего расшифровка работает полностью локально и без сети.
- выбор качества: поддержка моделей разного калибра — от tiny (~40mb) для быстрых набросков до medium (~750mb) для точного распознавания.
- запись с микрофона: можно надиктовать текст в реальном времени или просто перетащить (drag & drop) любой аудиофайл в окно.
- авто-транскрипция: начинает расшифровку сразу после добавления файла или остановки записи.
- интерфейс: темная тема, стеклянные карты (glassmorphism), шрифт Montserrat, плавная анимация прогресса, быстрый экспорт в .txt и копирование в один клик.

## запуск

просто открой `index.html` локально. для корректного кэширования моделей и работы динамических тем через ColorThief рекомендуется запускать через любой локальный веб-сервер (например, `python -m http.server 8000`).
